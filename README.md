# World Series of Balatro Seeding
Tool used to create a .csv file for TourneyBot seeding in World Series of Balatro

## How to use:
1. Make sure you have the requests library installed. To install it, `run pip install requests`
2. Run the code. It will create a file called `main.csv` in the folder it's ran in.
3. Upload the created file to Tourney Bot leaderboard using the command `/leaderboard import main [main.csv]`. It will take up to 20 minutes to process the file, as it has about 9000 entries.
